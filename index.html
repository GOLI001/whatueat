Создаём
Telegram Бота

Пошаговый гайд — от нуля до работающего бота за 15 минут

Python 3.10+ aiogram 3.x Telegram Bot API async / await
1
Создаём бота в BotFather
🚀

BotFather — официальный бот Telegram. Через него ты регистрируешь бота и получаешь токен доступа.

  • 1Открой Telegram и найди @BotFather
  • 2Отправь команду /start
  • 3Затем отправь /newbot
  • 4Придумай имя бота (любое, например: My Cool Bot)
  • 5Придумай username — должен заканчиваться на bot, например: my_cool_bot
⚠️
Важно! В конце BotFather выдаст TOKEN — что-то вроде 123456789:ABCdef...
Сохрани его — это ключ к твоему боту. Никому не давай!
2
Устанавливаем библиотеку
🧠

Используем aiogram — самую популярную асинхронную библиотеку для Telegram Bot API на Python.

terminal 
$ pip install aiogram
3
Первый код бота
🧱

Создай файл bot.py и вставь этот код:

bot.py 
import asyncio
from aiogram import Bot, Dispatcher, types
from aiogram.filters import Command

TOKEN = "ТВОЙ_ТОКЕН_СЮДА"

bot = Bot(token=TOKEN)
dp = Dispatcher()

@dp.message(Command("start"))
async def start_handler(message: types.Message):
    await message.answer("Привет! Я твой первый бот 😎")

async def main():
    await dp.start_polling(bot)

if __name__ == "__main__":
    asyncio.run(main())
4
Запускаем бота
▶️

В терминале запусти файл:

terminal 
$ python bot.py
  • 1Открой Telegram и найди своего бота
  • 2Напиши ему /start
  • 3Бот должен ответить — готово! 🎉
✅
Если бот ответил — поздравляю, ты только что создал своего первого Telegram бота!
5
Добавляем команды
⚡

Добавь обработчик — бот будет повторять любые сообщения. Вставь этот код до функции main():

bot.py 
@dp.message()
async def echo(message: types.Message):
    await message.answer(f"Ты написал: {message.text}")
💡
Обработчик без фильтров должен быть последним — иначе он перехватит все команды раньше них.
